Code of Conduct

We are committed to providing a friendly, safe, and welcoming environment for all.

Expected behavior
- Be respectful and considerate in comments, issues, PRs, and other communications.
- Assume good faith and be constructive when suggesting changes.

Unacceptable behavior
- Harassment, hate speech, or discriminatory language of any kind.
- Personal attacks, threats, or doxxing.

Enforcement
If you experience or witness unacceptable behavior, please report it to the maintainers via the project contact email listed in repository metadata. Reports will be handled confidentially.

This project adopts the Contributor Covenant principles; please act accordingly.
