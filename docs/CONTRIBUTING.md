Thank you for contributing to Omni.

Quick start
- Build: `cargo build --workspace`
- Format: `cargo fmt --all`
- Run Stage0 CLI: `cargo run -p omni-stage0 -- --help` or `cargo run -p omni-stage0 -- <command>`
- Tests: `cargo test --workspace`

How to contribute
- Open an issue to discuss larger changes before implementing.
- Make small, focused PRs with a clear description and tests when possible.
- Follow the repository formatting and linting (`cargo fmt`).
- Reference relevant examples under `examples/` and update `IMPLEMENTATION_STATUS.md` when adding features.

PR checklist
- [ ] Tests added or existing tests updated
- [ ] `cargo fmt` run
- [ ] Update `IMPLEMENTATION_STATUS.md` and `execution_log.md` if the change affects project status
- [ ] Link to an issue or include rationale in the PR description

Conduct
Please follow the project Code of Conduct: see CODE_OF_CONDUCT.md.

Contact
If you need to contact the maintainers privately, use the author email listed in the repository metadata.
